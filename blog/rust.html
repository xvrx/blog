  • install rust : https://www.rust-lang.org/tools/install

  • rust package manager = cargo

  • initiate rust project + also initiate git folder

new project
  • new rust project
cargo new <projectname>

or (without git initialization)

cargo new --vcs=git

cargo.toml contains the setting file for the project (dependencies etc)

  • run project (on directory)
cargo run
  • run quickly through the code to check for errors
cargo check
  • create a file cargo.lock that contains dependencies
  • create a file new dir 'target' that contains executable inside debug folder
cargo build
  • rust format (remove trail space / format code)
rustfmt <filename>
  • more info about rust cmd line
cargo help
